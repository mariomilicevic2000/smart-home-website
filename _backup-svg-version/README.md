# SVG-art version (archived)

This is the original fully procedural / hand-drawn SVG version of the Bonaca
hero scene and supporting art, before swapping in real photographic assets.

Kept for reference. Not imported anywhere, so it does not affect the build.

- `sections/Hero.tsx` — cinematic scene built from SVG (stone wall, olive
  branches, stone house, green shutters, AC, interior).
- `art/StoneWall.tsx` — procedural coursed-rubble masonry generator.
- `art/OliveBranch.tsx` — procedural olive sprig.
- `art/BonacaDevice.tsx` — the device (this one is kept in the live version too).
- `sections/ProductShowcase.tsx`, `sections/StoneQuote.tsx`, `styles.css`.

To restore: copy these back over `src/components/...` and `src/styles.css`.
